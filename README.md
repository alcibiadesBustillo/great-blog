# great-blog
Creating a blog


# Functionalities

1. Post article.
2. Recommend article via mail.
3. Add comment to article.
4. Tag function:
    - Find articles by tag.
    - Retrieving posts by similarity.
5. Custom template tags and filters:
    - Total posts.
    - Show latest post.
    - Markdown filter.
6. Adding a sitemap to the blog site.
